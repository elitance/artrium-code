## How-to
- Rename your file by double-clicking on the file
- Delete your file/tab by right-clicking on the file

## Shortcut Keys
- **⌘ + S**: Save file
- **⌥ + Number**: Open nth file
- **⌥ + W**: Delete tab
- **⌥ + N**: Switch to next tab

Note for Windows: ⌘ (Control), ⌥ (Alt)
