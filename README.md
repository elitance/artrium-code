## Shortcut Keys
- **⌘ + S**: Save file
- **⌘ + Space**: Enable Autocomplete
- **⌘ + /**: Toggle Line Comment
- **⌘ + `**: Toggle Terminal
- **⌥ + Number**: Open nth file
- **⌥ + W**: Delete tab
- **⌥ + N**: Switch to next tab
- **⇧ + F5**: Run File (JavaScript Only)

Note for Windows: ⌘ (Control), ⌥ (Alt)
